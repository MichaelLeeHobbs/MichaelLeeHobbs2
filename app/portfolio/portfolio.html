FreeCodeCamp.com - API Projects

I combined all of the FreeCodeCamp.com - API Projects into one project, deployed them on AWS. I use PM2 to run and watch for file changes. I do not like the idea of having a site auto-updated unless in has been tested and scheduled during a non-busy time.
